## ✨ What is DeltaK_CV

DeltaK_CV is a simple yet powerful tool created for Grok that provides a clear, structured and transparent **credibility analysis** of any content.

**New v2.0** includes the section **“Differences between initial reading and DeltaK”** that explicitly shows where a narrative loses consistency, omits facts or distorts reality.

---

## 🛠️ How to use it (super easy)

### Method 1 – Fastest (recommended)
1. Upload **this image** to any Grok chat  
2. Simply type:  
   **`DeltaK_CV:`**

Grok will automatically read the image and activate the fact-check.

### Method 2 – Direct text
In any Grok chat write:

DeltaK_CV: [paste the text, link or description of the post/article here]

## What DeltaK_CV returns (v2.0)

**✅ DeltaK_CV Activated**

**Overall Coherence Value (CV):** XX/100

### Quick Breakdown:
- Strengths and weaknesses at a glance

### Detailed Analysis (live fact-check)
- Point-by-point verification with real-time sources

### Differences between initial reading and DeltaK
- What seemed coherent at first glance  
- Where it loses consistency, omits information or distorts facts after full verification

### Final DeltaK Verdict
Clear and transparent judgment

---

## Features

- Analyzes X posts, articles, links, scientific papers and images
- AI-generated image detection (when present)
- Real-time verified sources
- Completely free and accessible to all Grok users
- No subscription required
